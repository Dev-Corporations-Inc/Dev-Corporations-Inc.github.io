Currently, If you have Mac/Unix/Linux, YOU DO NEED PYTHON >= 3.11 <= 4.0. You can install python at The Python Downloads Page,and download the latest version available for your OS
Most Linux distributions do have python pre-installed.
If your version of python is outdated on Linux, you can follow these commands:
Ubuntu/Debian based systems: sudo apt update sudo apt install python3.11
For Fedora: sudo dnf install python3.11
